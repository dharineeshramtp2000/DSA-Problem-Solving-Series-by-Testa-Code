
### You are given an input array of integers ```nums```. You need to return the same array by reversing its integers.

### Constraint:
 - Time - O(n)
 - Space - O(1)

### Sample Input 1:
```
language = [1,7,9,6,5,4]

````
### Sample Output 1:
```
[4,5,6,9,7,1]
```

### Sample Input 2:
```
language = [10]

````
### Sample Output 2:
```
[10]
```
